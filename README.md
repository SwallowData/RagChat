# RagChat  
A rag item that you can talk to  
本项目使用了智谱进行词向量化，需要在.env上添加你的key  
创建虚拟环境  
'''
conda create -n RagChat python=3.11
'''
启动虚拟环境  
'''
conda activate RagChat
'''
添加对应的环境  
'''
pip install -r requirements.txt
'''
运行程序  
'''
cd code
streamlit run main.py
'''
